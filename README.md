# Algorithm

- WSL2
- Visual Studio Code
- gcc 10 or above in PATH
- CPython3 or PyPy3 in PATH

## Install Dependencies

```bash
pip insall -r requirements.txt
```

## Validator

```bash
python validate.py <code path> # only support C++ and Python3
```

## Minifier

```bash
python minify.py <code path> # only support Python3
```
